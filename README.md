# A Knight in Space

A clicker-like game based on the concept of a Knight abandoned in space.

## Team Members

Albin Shrestha, Zac Galer, Connor Kite, and Mark Cheng

## How to Run

Open project in Xcode and run using an iPhone 12 Pro simulator
